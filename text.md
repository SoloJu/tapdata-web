1.目录结构
build  ----------------- webpack相关配置文件
config ----------------- vue基本配置文件
node_modules ----------- 依赖包
src -------------------- 项目核心文件
  assets --------------- 静态资源（样式类文件）
  commponents ---------- 公共组件
  router --------------- 路由
  i18n ----------------- 国际化
  view ----------------- vue页面
  vuex ----------------- 状态管理
  main.js -------------- 入口文件
  App.vue -------------- 根组件
static ----------------- 静态资源（放图片）
