# tapdaas

> Data as a service

## 快速开始

1. **安装依赖**
``` bash
yarn install
```
2. **启动应用**
```bash
yarn start 启动并打开应用
// or
yarn start --dev 开启dev环境的接口代理
```

3. **打包**
```bash
yarn build
```
