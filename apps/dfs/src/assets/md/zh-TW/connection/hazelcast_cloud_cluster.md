## **連接配置幫助**

### **1. Hazelcast Cloud 安裝說明**

請遵循以下說明以確保在 Tapdata 中成功添加和使用Hazelcast Cloud數據庫。

Tapdata系統支持 hazelcast cloud 作為目標。
